El uso de la terminal es una habilidad fundamental para cualquier desarrollador, ya que permite gestionar archivos y proyectos de manera más ágil que la interfaz gráfica. Al trabajar en entornos de programación, muchas herramientas y servidores no cuentan con iconos, por lo que saber navegar mediante comandos es la única forma de interactuar con ellos. Además, la terminal facilita la automatización de tareas repetitivas y es indispensable para utilizar sistemas de control de versiones como Git de forma profesional. Entender la jerarquía de carpetas y saber moverse entre ellas previene errores al ejecutar scripts o instalar dependencias. También permite tener un control total sobre los permisos y la estructura del proyecto sin depender del mouse. En definitiva, la terminal no solo mejora la productividad, sino que es el lenguaje base para administrar cualquier sistema de software moderno.
